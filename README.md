
### CRM - Customer Relationship Management

#### Getting Started

Install gems:

    $ bundle install

Migrate database:

    bundle exec rake db:setup

Run tests:

    bundle exec rake

Run:

    $ bundle exec rails s

Open [http://localhost:3000](http://localhost:3000).  w0oh0o! :)
